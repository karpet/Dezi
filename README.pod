=head1 Dezi

Dezi is a search platform based on Apache Lucy, Swish3, Plack and Search::OpenSearch.
Dezi integrates several search toolkits into an easy-to-use server.

=head2 INSTALLATION

To install this module, run the following commands:

	perl Makefile.PL
	make
	make test
	make install

=head2 SUPPORT AND DOCUMENTATION

After installing, you can find documentation for this module with the
